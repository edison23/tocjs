TOCjs - Javascript generator of a table of contents from HTML document

Insert your HTML source code of your document into the field below and click the button Generate TOC. This generates nested ordered lists of the headings found in the document. The items in the lists are links to the headings. These links use IDs of the headings which are either added to each heading tag if there was none before, or are used the existing ones if there was ID already.

In the output section, you will find the generated TOC rendered as HTML as well as inserted into the textarea as a HTML source and in the textarea below, you'll find your document with the TOC and with the IDs added to the document (if there were any heading tags missing the ID attribute).

To use the output of this tool, copy the source code of your document with the heading IDs back to where you use your document, or just add the source of the generate TOC to your document in case you had all the IDs in the document already.
